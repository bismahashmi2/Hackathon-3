# Lab 14-01: Capstone Design

## Objectives
- Design complete robotics system
- Specify requirements and architecture
- Plan implementation and testing

## Implementation
Students propose capstone project integrating multiple course modules. Design document includes system architecture, component specifications, testing plan, and success criteria.

## Rubric: 100 points (Design: 40/Feasibility: 30/Documentation: 30)
